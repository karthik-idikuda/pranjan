# PRAJNA

Modular AI system for satellite data processing with graph-based pipeline architecture, configurable model training, and formal verification. Developed for ISTRAC/ISRO research applications.

## Features

- Graph-based processing pipeline
- Configurable model training with YAML definitions
- Satellite telemetry data ingestion
- Formal verification of processing outputs
- Extensible module architecture
- Comprehensive test suite

## Tech Stack

Python, NumPy, SciPy, YAML, Graph Processing

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/pranjan.git
cd pranjan
pip install -e .
```

### Usage

```bash
python -m prajna --config config/default.yaml
```
